# 阶段五：前端界面 实现计划

**Goal:** 开发用户界面，实现文档上传、审核进度展示、结果展示和报告导出功能。

**Architecture:** 使用 React + TypeScript，构建轻量级 SPA 应用，通过 FastAPI 后端调用 REST API。

**Tech Stack:** React 18, TypeScript, Vite, Tailwind CSS, React Query

---

## 核心功能

| 功能 | 描述 |
|------|------|
| **文档管理** | 待审核文档上传、列表、删除 |
| **知识库管理** | 知识库列表、文档导入 |
| **审核流程** | 创建审核任务、查看进度 |
| **结果展示** | 审核问题列表、详情 |
| **报告导出** | 导出审核报告（JSON/HTML） |

---

## 页面结构

```
/knowledge-bases      # 知识库列表
/knowledge-bases/:id   # 知识库详情
/audit                # 待审核文档列表
/audit/:id            # 文档详情
/audit/:id/result     # 审核结果
```
